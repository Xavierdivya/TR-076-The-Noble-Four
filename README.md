# TR-076-The-Noble-Four

## 📌 Special Needs Education Plan Generator - EduPath


## 📖 Description
EduPath is an AI-powered React web application for generating personalized education plans (IEPs) for children with special needs. It uses a JavaScript-based mock backend, MongoDB schema design, and Claude AI integration to support assessment tracking and parent–educator collaboration.

---

## 🚀 Features
- AI-based IEP generation
- Parent and educator collaboration
- Student assessment tracking
- Progress monitoring
- Personalized recommendations

---

## 🛠️ Tech Stack
- Frontend: React
- Backend: JavaScript (Mock Backend)
- Database: MongoDB (Schema Design)
- AI Integration: Claude API
